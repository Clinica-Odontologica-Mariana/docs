# Protótipo Alta Fidelidade - Mobile

Este protótipo de alta fidelidade representa a versão mobile da interface do sistema,
adaptada para dispositivos de tela reduzida com foco em usabilidade e acessibilidade.

O design foi pensado para garantir uma experiência fluida e intuitiva em smartphones,
respeitando as diretrizes de interface para plataformas móveis e as necessidades
identificadas durante o levantamento de requisitos.
