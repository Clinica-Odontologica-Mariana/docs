# Protótipo Alta Fidelidade - Desktop

Este protótipo de alta fidelidade representa a versão desktop da interface do sistema,
desenvolvido com base nos requisitos levantados e nas histórias de usuário definidas
ao longo do projeto.

A navegação interativa permite visualizar o fluxo completo da aplicação, incluindo
as principais telas, componentes visuais e interações planejadas para a experiência
do usuário em telas maiores.

> Para interagir com o protótipo, utilize o painel abaixo. Clique nos elementos
> para navegar entre as telas.
